# 图纸列表、排序与混合参数搜索改造设计

## 目标

- 图纸识别页只承担 DXF 上传，减少重复入口和无关信息。
- 图纸列表在常用桌面宽度内完整显示，不需要横向拖动才能操作。
- 图纸和成品型号按字母、数字自然顺序排列。
- 齿型、齿数和模数支持字母数字混合查询。

## 页面调整

### 图纸识别

- 合并单张上传和批量导入，使用一个支持多选和拖拽的 DXF 上传区。
- 选择 1 至 50 张文件后统一提交到批量上传接口，并显示逐文件结果。
- 删除本页搜索表单和图纸记录表格。
- 删除已经不再使用的批量高清预览入口。
- 保留“待确认图纸”和“已确认图纸”导航。

### 待确认与已确认图纸

- 列表只显示产品编号、产品名称、分类/材质、版本、状态和操作。
- 厚度、尺寸、齿轮参数、备注等完整数据继续保留在图纸详情页。
- 已确认图纸保留参数搜索和 Excel 导出。
- 搜索控件自动换行，不撑宽页面。
- 操作列保持可见，表格内容可换行，不产生页面级横向滚动。

### 其他页面

- 检查成品、板料、余料及流水页面的宽表。
- 汇总列表优先合并相关字段、允许文本换行；明细和流水不删除业务信息。
- 通用表格样式限制在内容区域宽度内，操作列保持可见。

## 排序规则

- 新增统一的自然排序键，忽略英文字母大小写，并将连续数字按数值比较。
- 示例顺序：`A2`、`A10`、`B1`，而不是 `A10`、`A2`、`B1`。
- 应用于待确认图纸、已确认图纸、图纸选择项、成品库存汇总和成品出库列表。
- 相同型号的不同版本仍按版本号从高到低排列。

## 搜索规则

- 普通图纸搜索继续匹配编号、名称、分类、备注、材质等文本字段。
- 对 `OT48`、`OT48(52)` 等齿数查询，识别 `IT/IL/IR/OT/OL/OR` 前缀，并同时匹配齿型和齿数文本。
- 只输入 `OT` 时匹配齿型，只输入 `48` 时匹配齿数。
- 模数按完整文本匹配，支持 `DP`、`DP15` 等英文字母与数字组合。
- 相同规则应用于页面筛选、计划图纸查询和 AI 图纸查询。
- 成品、板料和余料现有文本输入继续使用不区分大小写的包含匹配，确保混合型号可查询。

## 验证

- 测试上传页只有一个多文件上传表单，且不再包含搜索和图纸记录。
- 测试待确认和已确认列表的精简列及自然排序。
- 测试 `OT48`、`OT48(52)`、`DP15` 的页面和 AI 查询结果。
- 测试成品库存及成品出库按自然顺序显示。
- 在桌面和窄屏视口检查关键页面没有页面级横向滚动，操作按钮无需左右拖动即可看到。
